Investigating Bandgap-Volume Correlations in Kesterite Solar Materials

Overview
This project focuses on the computational analysis of Kesterite (CZTS) materials to identify optimal candidates for photovoltaic applications. By leveraging the Materials Project API (MP-API), I retrieved crystal data for quaternary semiconductors containing Cu, Zn, Sn, and S to analyze how structural properties influence electronic bandgaps.

Key Insights

Data Retrieval: Programmatically queried over [Number] materials from the Materials Project database.

Correlations: Identified structural trends between unit cell volume and optical bandgap energy.

Research Context: This screening was used to provide theoretical context for my experimental work on solution-processed CZTS thin films.

Tools Used

Python: (Pandas, Matplotlib, Seaborn)

Materials Informatics: Materials Project API (MP-API)

Physics Principles: Vegard’s Law, Shockley-Queisser Limit
